# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a Rust training materials repository containing **7 interconnected books** written in Markdown and built with [mdBook](https://rust-lang.github.io/mdBook/). The books form a structured curriculum from language basics through advanced/expert topics.

## Building the Books

Prerequisites:
```bash
cargo install mdbook mdbook-mermaid
```

Build commands (run from repository root):
```bash
cargo xtask build    # Build all books into site/ (local preview)
cargo xtask serve    # Build and serve at http://localhost:3000
cargo xtask deploy   # Build into docs/ (GitHub Pages deployment)
cargo xtask clean    # Remove site/ and docs/
```

To work on a single book:
```bash
cd <book-dir> && mdbook serve --open
```

## Repository Structure

```
*book/              # One directory per book (mdbook project)
  book.toml         # mdbook configuration
  src/
    SUMMARY.md      # Chapter ordering/navigation
    ch*.md          # Chapter content

xtask/              # Build orchestration
  src/main.rs       # cargo xtask implementation (builds all books + landing page)

.github/workflows/  # CI/CD (auto-deploys to GitHub Pages on push to master)
```

## Architecture Notes

- `xtask/src/main.rs` defines the `BOOKS` constant listing all books with their slugs, titles, and categories — add new books here when creating them
- Each book directory is an independent mdBook project with its own `book.toml`
- The landing page (`index.html`) is auto-generated by xtask to link all books
- All books use the Mermaid preprocessor for diagrams via `mermaid.min.js` and `mermaid-init.js`
